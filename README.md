# Cow Crack
a tool to "crack" facebook account.
# install packpages
```BASH
pkg install git
pkg install python
```
## clone
```BASH
git clone https://github.com/Latip176/cow_crack
```
## setup
```BASH
cd cow_crack
python3 -m pip install -r requirements.txt
python setup.py build_ext --inplace
```
# running
```BASH
cd $HOME/cow_crack
python run.py
```
# command buat update
```BASH
cd $HOME/cow_crack
git pull
rm -rf *.so
python setup.py build_ext --inplace
python run.py
```
